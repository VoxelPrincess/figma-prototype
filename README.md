# Figma Prototype

## Overview

This repository contains a responsive and interactive portfolio prototype designed in Figma. The prototype highlights skills, projects, and personal branding with a focus on usability and accessibility.

## Features

- **Responsive Design**: Includes layouts for both desktop and mobile devices.
- **Interactive Elements**: Navigation and components with clickable interactions.
- **Consistent Style**: Aligned with the style guide using predefined colors, typography, and icons.

## Pages Included

1. **Home**: Introduction with visual highlights.
2. **About Me**: Personal background and career goals.
3. **My Skills**: A detailed overview of technical competencies.
4. **Portfolio**: Showcase of projects and achievements.
5. **Contact**: Communication details with a form.

## Usage

- Open the [Figma Prototype](https://www.figma.com/design/I2ieEZDdSiRGoNzEWCgV9z/The-first-version-of-the-prototype?t=0o4zagXMwZ2gnilt-1) to explore the interactive design.
- Navigate through desktop and mobile flows to explore features.

## Installation

1. Clone this repository.
2. Open the prototype in Figma to view or edit.
